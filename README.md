# 手写 mini-webpack mini-vite

## mini-webpack 项目启动

```sh
cd  mini-webpack
```

```sh
yarn
```

```sh
yarn  build
```

## mini-vite 项目启动

```sh
cd  mini-webpack
```

```sh
yarn
```

```sh
yarn  dev
```
